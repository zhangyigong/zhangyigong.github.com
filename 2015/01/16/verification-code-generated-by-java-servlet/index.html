使用servlet 生成验证码

为什么要使用验证码

输入验证码是一个很麻烦的一个过程，很多用户都不喜欢，但是验证码却总是存在于各类的网站和Web 应用中，比如

  • 回复帖子的时候
  • 申请账号的时候
  • 用户登录的时候
    …

为什么要使用验证码呢？

因为计算机可以模拟人的行为对网站进行一系列的操作，如果不使用验证码，将会导致一些可怕的事情发生

  • 对特定用户不断登录破解密码
  • 对某个网站不断创建无效账户
  • 对某个网站提交垃圾数据
  • 对网站投票进行刷票
    …
